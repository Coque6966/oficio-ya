# Guía de Acceso - OficioYa 🚀

**¡TU APP YA ESTÁ EN VIVO!**
Accede aquí desde tu PC o Celular: [https://oficio-ya.vercel.app](https://oficio-ya.vercel.app)

---

Esta guía te permite probar todas las funciones de la plataforma en sus diferentes roles (Cliente, Trabajador, Administrador) en el entorno de producción.

## 1. Acceso como Cliente (Usuario)
1. Ve a la página de inicio: [https://oficio-ya.vercel.app](https://oficio-ya.vercel.app)
2. Haz clic en "Inicia Sesión" o "Regístrate".
3. Puedes usar cualquier correo electrónico.
4. **Función principal**: Buscar profesionales por categoría, ver perfiles y **agendar un servicio**.

## 2. Acceso como Trabajador (Proveedor)
1. Una vez que hayas iniciado sesión como usuario, ve a la sección ["Soy Profesional"](https://oficio-ya.vercel.app/pricing).
2. Elige un plan (puedes probar primero el **Plan Gratis**).
3. Completa tu perfil en el Dashboard de Proveedor.
4. **Reglas de Monetización**:
   - **Plan Gratis**: Tienes **2 trabajos gratis**. Después de eso, el sistema te pedirá actualizar.
   - **Plan Básico ($99)**: Hasta 7 trabajos al mes.
   - **Plan Pro ($199)**: Sin límites.

## 3. Acceso como Administrador
Para gestionar la plataforma como Administrador:
1. Entra a [https://oficio-ya.vercel.app/admin](https://oficio-ya.vercel.app/admin)
2. Debes tener los permisos correspondientes. Si necesitas hacertadmin localmente, usa `npm run admin -- tu-correo@ejemplo.com`.
